# Netflix Clone

## Project Overview
This project is a simple clone of the Netflix landing page that showcases the streaming platform's key features. It is built using HTML and features Tailwind CSS for styling, providing a mobile-responsive design that captures the essence of Netflix's web interface.

## Installation
To run the Netflix Clone project locally, follow these steps:

1. **Download or clone the repository**:
   ```bash
   git clone <repository-url>
   cd netflix-clone
   ```

2. **Open the `index.html` file**:
   You can do this using a web browser of your choice. Simply double-click the `index.html` file to view the project.

3. **Required Libraries**:
   The project uses external libraries linked in the HTML, so no additional installation is necessary beyond having an active internet connection.

## Usage
Once the project is open in your browser, you can interact with the following features:

- **Hero Section**: Contains a sign-in prompt and a subtitle describing Netflix's offerings.
- **Content Sections**: Displays information about watching Netflix on various devices.
- **Footer**: Features links for customer support, account information, and language selection.

## Features
- Responsive design that adjusts to various screen sizes.
- Eye-catching hero section with a background image and overlay text.
- Integration of Font Awesome for icons.
- Display of a promotional video within the content section.
- Functional navigation links and language selection dropdown.

## Dependencies
This project uses the following external dependencies as indicated in the `index.html` file:
- [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework for styling.
- [Font Awesome](https://fontawesome.com/) - A library for icons and social logos.

## Project Structure
The project directory contains the following structure:

```
/netflix-clone
├── index.html
```

- **index.html**: The main file that contains all the HTML markup, styling, and links to external resources used for the Netflix clone.
